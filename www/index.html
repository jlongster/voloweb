volo
Are you tired of repeating yourself when building web sites? Volo is a tool which lets you quickly create projects, add libraries, build them into optimized formats, and deploy them. All with a few keystrokes.
Just type volo <command> and you'll be able to:
  • Quickly add javascript libraries: Just tell volo the library name or a raw URL, and volo will find it, add it and all dependencies, and convert it into an AMD module if needed.
  • Automatically compile Less and Coffeescript: Volo will watch your files for changes and automatically compile them. This is builtin to volo's development server. (TODO)
  • Fire up a development server: View your site over HTTP if you don't already have a server-side component set up. This also compiles your Less and Coffeescript for you.
  • Build your site optimally: Concatenate and minify all your CSS and js with one build command. Volo uses r.js to flatten all your AMD modules and CSS files and minimize them.
  • Deploy straight to github: If your site is all client-side, you can publish to github with one command. Your site is live within seconds.
  • Manage the appcache: Make or update an appcache with one simple command. Never worry about adding files or changing the appcache version again.

It's modular, too!

(work in progress)

Add your own commands easily, and install commands from the community. Every volo project has a file called volofile at the root which is just a list of commands implemented in javascript. You can easily add your own commands.

Volo commands are also available via npm. For example, to install the ghdeploy command, simply run npm install volo-ghdeploy.
